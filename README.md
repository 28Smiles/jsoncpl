# jsoncpl

Compare and Lint Json Files

```
USAGE:
    jsoncpl.exe [OPTIONS] [FOLDERS]...

ARGS:
    <FOLDERS>...    

OPTIONS:
        --format               
    -h, --help                 Print help information
        --indent <INDENT>      [default: 4]
    -o, --order <ORDER>        [default: asc] [possible values: asc, desc]
    -s, --sort <SORT>          [default: default] [possible values: natural, default]
        --skip-check-order     
        --skip-check-parity    
        --skip-check-style
```
